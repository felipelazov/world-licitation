# Story 4.4 — Campo de Validade + Status Automatico

> **Epic:** 4 — Repositorio de Documentos
> **Prioridade:** P0 | **Estimativa:** 0.5 dia | **Depende de:** 4.2

---

## Acceptance Criteria

- [ ] Campo obrigatorio `expires_at` ao subir documento
- [ ] Status calculado automaticamente: valido (verde), vencendo (amarelo), vencido (vermelho), pendente (cinza)
- [ ] "Vencendo" = dentro do periodo de alerta (configuravel, default 30 dias)
- [ ] Badge de status no card do documento
- [ ] Dashboard de docs: barra de progresso mostrando % validos vs total
- [ ] Cron diario que recalcula status de todos os documentos
