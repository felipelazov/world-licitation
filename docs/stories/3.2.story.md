# Story 3.2 — Formulario de Analise GO/NO-GO

> **Epic:** 3 — Analise GO/NO-GO
> **Prioridade:** P0 | **Estimativa:** 1-2 dias | **Depende de:** 3.1, 2.7

---

## Acceptance Criteria

- [ ] Formulario com 10 criterios (5 eliminatorios sim/nao + 5 scoring 1-5)
- [ ] Criterios eliminatorios: objeto, quantidade, local, prazo, habilitacao
- [ ] Criterios scoring: margem, clausulas, historico orgao, concorrencia, risco/retorno
- [ ] Campo de observacao por criterio
- [ ] Se qualquer eliminatorio = NO → decisao automatica NO-GO
- [ ] Score calculado automaticamente (media dos scoring)
- [ ] Recomendacao visual: GO (verde), NO-GO (vermelho), GO com ressalvas (amarelo)
- [ ] Campo de justificativa geral
- [ ] Salva e atualiza status do edital automaticamente
- [ ] Acessivel direto da pagina de detalhe do edital
