# Story 8.1 — Wizard de Setup

> **Epic:** 8 — Onboarding
> **Prioridade:** P0 | **Estimativa:** 1-2 dias | **Depende de:** 1.4, 1.5

---

## Acceptance Criteria

- [ ] Wizard multi-step apos primeiro login
- [ ] Step 1: Dados da empresa (razao social, CNPJ, segmento, regiao, capacidade)
- [ ] Step 2: Configurar filtros de monitoramento (palavras-chave, regiao)
- [ ] Step 3: Upload dos primeiros documentos (certificado digital, alvara, certidoes)
- [ ] Step 4: Definir planilha de custo base (usar template ou criar do zero)
- [ ] Progresso visual: stepper com % completo
- [ ] Pode pular steps (nao obrigatorio terminar tudo)
- [ ] Ao concluir: redirect pro dashboard com mensagem de boas-vindas
- [ ] Flag `onboarding_completed` no perfil (nao mostra wizard de novo)
