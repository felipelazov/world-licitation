## Descricao

Implementar scraper que monitora editais de pregao eletronico no portal Compras.gov.br (federal). Usar API publica onde disponivel + headless browser como fallback.

## Acceptance Criteria

- [x] Scraper busca editais de pregao eletronico no Compras.gov.br
- [x] Extrai: numero, orgao, objeto, valor estimado, data sessao, UASG, link
- [x] Filtra por palavras-chave configuradas pelo tenant
- [x] Salva editais novos no banco (upsert por external_id) — via upsertEdital query
- [ ] Download do PDF do edital (salva no Supabase Storage) — V2
- [x] Log de execucao (sucesso, erros, editais encontrados)
- [x] Retry automatico em caso de falha (max 3 tentativas)
- [x] Rate limiting pra nao sobrecarregar o portal
