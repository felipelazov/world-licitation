# Story 6.2 — Painel de Sessao do Pregao

> **Epic:** 6 — Assistente de Pregao
> **Prioridade:** P0 | **Estimativa:** 2 dias | **Depende de:** 6.1, 5.3

---

## Acceptance Criteria

- [ ] Tela dedicada pra acompanhar sessao de pregao
- [ ] Header: numero do pregao, orgao, hora da sessao
- [ ] Piso de margem visivel e destacado (vermelho se proximo)
- [ ] Calculadora ao vivo: input de valor → mostra margem resultante (%)
- [ ] Input de lances dados (valor + timestamp)
- [ ] Log de lances (meus e concorrentes se visivel)
- [ ] Timer da sessao
- [ ] Alerta visual quando preco atinge zona de piso
- [ ] Indicador de margem: barra colorida (verde → amarelo → vermelho)
- [ ] Dados do edital e planilha de custo acessiveis na mesma tela (sidebar ou tabs)
