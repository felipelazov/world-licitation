# Story 4.2 — Upload e Visualizacao de Documentos

> **Epic:** 4 — Repositorio de Documentos
> **Prioridade:** P0 | **Estimativa:** 1 dia | **Depende de:** 4.1, 1.5

---

## Acceptance Criteria

- [ ] Upload via drag-and-drop ou click (PDF, JPG, PNG)
- [ ] Limite: 10MB por arquivo
- [ ] Upload pro Supabase Storage (bucket `documents`, path: `{tenant_id}/{category}/{filename}`)
- [ ] Visualizador inline: PDF viewer, imagem viewer
- [ ] Download do arquivo original
- [ ] Metadados: nome, categoria, data emissao, data vencimento, notas
- [ ] Delete com confirmacao
- [ ] Loading state durante upload
- [ ] Validacao de tipo de arquivo
