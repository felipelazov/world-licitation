## Descricao

Criar o layout principal da aplicacao web: sidebar com navegacao, header com user info, tema claro/escuro e responsividade.

## Acceptance Criteria

- [x] Layout com sidebar colapsavel (icone + texto / so icone)
- [x] Sidebar com navegacao dos modulos do MVP
- [x] Header com: nome da empresa, nome do usuario, avatar, dropdown (perfil, sair)
- [x] Tema claro e escuro (toggle no header)
- [x] Responsivo: sidebar vira drawer em mobile
- [x] Breadcrumbs nas paginas internas
- [x] Pagina de dashboard (placeholder com cards vazios)
- [x] Loading states e skeletons
- [x] 404 page
- [x] Favicon e meta tags

## Design System Base

| Token | Valor |
|---|---|
| Font | Inter (corpo), JetBrains Mono (numeros/codigo) |
| Radius | 8px (cards), 6px (inputs), 4px (badges) |
| Spacing | 4px grid |
| Colors | shadcn/ui default (customizar depois) |
| Sidebar width | 256px (expandida), 64px (colapsada) |
