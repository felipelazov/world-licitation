# 🧠 Inicializar Board de Conselheiros

> Ativa automaticamente o conselho quando este arquivo é aberto

**AÇÃO AUTOMÁTICA:** Ativar advisor-chief na inicialização do projeto

---

## Sistema Automático de Inicialização

Este projeto está configurado para ativar seu **Board de Conselheiros** automaticamente.

Quando você abre este projeto e faz qualquer pergunta, o conselho responde:

- 💡 **Simon Sinek** — Propósito, liderança, visão
- 💰 **Alex Hormozi** — Monetização, ofertas, números
- 🍎 **Steve Jobs** — Produto, design, simplicidade

---

## Para Ativar Manualmente

Se precisar reconectar com o conselho a qualquer momento:

```
/squad-advisor:agents:advisor-chief
```

Ou use o atalho:

```
/conselho
```
