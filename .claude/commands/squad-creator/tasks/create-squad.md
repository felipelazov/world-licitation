# Task: Create Squad — Criar Squad Completo de IA

**Task ID:** squad-creator/create-squad
**Version:** 1.0.0
**Execution Type:** Hybrid (Incremental ou YOLO)
**Duration:** 30-90 minutos (depende do dominio)
**Agent:** squad-chief

---

## Purpose

Criar um squad completo e production-ready de agentes IA especializados em qualquer dominio. O squad e construido com base em mentes reais de elite — pesquisadas, clonadas e validadas.

Pipeline: Discovery → Research → Architecture → Creation → Integration → Validation

---

## How to Use

Ao ativar esta task, o squad-chief sera ativado e executara a task `create-squad.md` do squad-creator.

Voce pode informar o dominio diretamente:

```
/squad-creator:tasks:create-squad vendas B2B
```

Ou ativar e responder as perguntas guiadas.

---

## Activation

Load and execute: `squads/squad-creator/tasks/create-squad.md`
With agent: `squads/squad-creator/agents/squad-chief.md`
