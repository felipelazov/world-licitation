# Auto-Activate Advisor Board

> This hook automatically activates the advisor-chief when the project opens

**ACTIVATION-COMMAND:** `/squad-advisor:agents:advisor-chief`

**TRIGGER:** on-project-open

---

Quando este projeto abre, o Board de Conselheiros é acionado automaticamente.

Você será conectado com Simon Sinek, Alex Hormozi e Steve Jobs.

Comece descrevendo sua situação ou pergunta!
