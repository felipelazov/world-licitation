# Checklist GO/NO-GO — Triagem de Edital

## 5 Criterios Binarios

Todos devem ser GO para participar. Qualquer NO-GO elimina o edital.

### 1. Objeto Compativel?
- [ ] Fornecemos exatamente o que esta sendo pedido?
- [ ] Especificacao nao e restritiva (marca/modelo unico)?
- [ ] Quantidade compativel com nossa capacidade?

**Resultado:** GO / NO-GO

### 2. Habilitacao Atingivel?
- [ ] Temos TODOS os documentos exigidos?
- [ ] Certidoes estao dentro da validade?
- [ ] Atestado de capacidade tecnica atende o volume exigido?
- [ ] Alvara sanitario em dia?

**Resultado:** GO / NO-GO

### 3. Entrega Viavel?
- [ ] Local de entrega esta no nosso raio logistico?
- [ ] Prazo de entrega e factivel?
- [ ] Temos capacidade de producao/montagem no prazo?
- [ ] Logistica de transporte esta mapeada?

**Resultado:** GO / NO-GO

### 4. Preco com Margem?
- [ ] Preco referencia permite margem >= piso definido?
- [ ] Custos (materia-prima + frete + impostos) estao calculados?
- [ ] Margem liquida estimada >= 8%?

**Resultado:** GO / NO-GO

### 5. Risco Aceitavel?
- [ ] Penalidades por atraso sao proporcionais?
- [ ] Prazo de pagamento e aceitavel (30 dias max)?
- [ ] Nao ha garantia contratual excessiva (>5%)?
- [ ] Historico do orgao e positivo (paga em dia)?

**Resultado:** GO / NO-GO

---

## Veredicto Final

- 5/5 GO → **PARTICIPAR**
- 4/5 GO → **AVALIAR** (depende de qual criterio falhou)
- 3/5 ou menos → **NAO PARTICIPAR**
