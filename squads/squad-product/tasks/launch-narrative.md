## Overview

A narrativa de lancamento traduz positioning em messaging adaptado para cada audiencia. O mesmo produto precisa de mensagens diferentes para: clientes potenciais, clientes existentes, imprensa, parceiros, e time interno. A narrativa core e UMA — as adaptacoes sao muitas.

---

## Action Items

### Step 1: Definir Narrativa Core
- 1 paragrafo que captura: problema + insight + solucao + valor + prova
- Deve funcionar isolado (se alguem le so isso, entende tudo)
- Baseado diretamente no positioning canvas

### Step 2: Adaptar por Audiencia
- **Prospects:** Foco em problema + valor + prova social
- **Clientes existentes:** Foco em novidade + beneficio adicional + como usar
- **Imprensa/Analistas:** Foco em trend + categoria + diferencial + metricas
- **Parceiros:** Foco em oportunidade conjunta + complementaridade
- **Time interno:** Foco em visao + impacto + como vender/suportar

### Step 3: Definir Key Messages
- 3-5 mensagens-chave que aparecem em TODA comunicacao
- Cada mensagem: claim + evidence + example
- Consistentes mas adaptaveis ao tom de cada canal

### Step 4: Criar Messaging Toolkit
- One-liner (1 frase)
- Boilerplate (1 paragrafo)
- Key messages (3-5 com evidence)
- FAQ de positioning (perguntas que vao surgir)
- Messaging don'ts (o que NAO dizer)

### Step 5: Distribuir para Squads
- squad-marketing: messaging para campanhas
- squad-content-distillery: narrativa core para atomizar
- squad-sales: alinhamento pitch + launch message
- squad-customer-success: como comunicar para base existente

---

## Acceptance Criteria

1. Narrativa core clara e auto-suficiente (1 paragrafo)
2. Adaptacoes para pelo menos 3 audiencias
3. 3-5 key messages com evidence
4. Messaging toolkit completo (one-liner, boilerplate, FAQ, don'ts)
5. Distribuido para squads relevantes via sinapse

---

## Veto Conditions

- NAO lancar sem narrativa core definida — cada squad inventa sua propria e vira caos
- NAO adaptar mensagem sem manter consistencia com positioning — adaptacao != contradicao
- NAO usar jargao interno na comunicacao externa — se o cliente nao entende, nao funciona

---

## Handoff

- Messaging toolkit → squad-marketing (campanhas)
- Narrativa core → squad-content-distillery (atomizar em conteudo)
- Pitch alignment → squad-sales (consistencia)
- Base messaging → squad-customer-success (comunicar para clientes)
