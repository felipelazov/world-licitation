# Checklist de Completude do Genius Profile

**Proposito:** Validar que o perfil integrado de genialidade esta completo em todas as dimensoes, com cruzamentos entre frameworks validados e insights claros para o fundador.
**Agente responsavel:** zona-genialidade-chief (apos receber outputs dos 7 agentes)
**Padrao:** Todas as 6 dimensoes mapeadas + cruzamentos validados + narrativa coerente

---

## DIMENSAO 1: ZONA DE GENIALIDADE (Gay Hendricks)

- [ ] Zona predominante identificada (Incompetencia / Competencia / Excelencia / Genialidade)
- [ ] Zona de Genialidade descrita em linguagem especifica do fundador
- [ ] Upper Limit Problem ativo identificado e nomeado
- [ ] Armadilha da Zona de Excelencia analisada (se aplicavel)
- [ ] Big Leap path delineado com primeiros passos concretos

**Se incompleto:** Retornar ao agente gay-hendricks para aprofundamento.

---

## DIMENSAO 2: TALENTOS DOMINANTES (Don Clifton)

- [ ] Top 5 talentos CliftonStrengths identificados e ordenados
- [ ] Dominio de forca principal mapeado (Execucao / Influencia / Relacionamento / Pensamento Estrategico)
- [ ] Descricao de como cada talento se manifesta no dia-a-dia do fundador
- [ ] Talentos-sombra identificados (talentos que drenam quando mal aplicados)
- [ ] Combinacoes raras de talentos destacadas (se existirem)

**Se incompleto:** Retornar ao agente don-clifton para complementar analise.

---

## DIMENSAO 3: HABILIDADE UNICA (Dan Sullivan)

- [ ] Unique Ability statement formulado em 1-2 frases
- [ ] Atividades que geram energia superior identificadas (pelo menos 3)
- [ ] Atividades que drenam energia listadas para delegacao
- [ ] Visao 10x articulada (ambicao que parece impossivel mas e motivadora)
- [ ] "Who Not How" mapeado — quem o fundador precisa ao redor

**Se incompleto:** Retornar ao agente dan-sullivan para refinamento.

---

## DIMENSAO 4: PERFIL DE RIQUEZA (Roger Hamilton)

- [ ] Perfil Wealth Dynamics identificado (Creator / Star / Supporter / Deal Maker / Trader / Accumulator / Lord / Mechanic)
- [ ] Estrategia de fluxo de valor alinhada ao perfil
- [ ] Nivel no Wealth Spectrum estimado (Infravermelho a Ultravioleta)
- [ ] Alavancas naturais de riqueza do perfil descritas
- [ ] Armadilhas do perfil documentadas (o que evitar)

**Se incompleto:** Retornar ao agente roger-hamilton para complementar.

---

## DIMENSAO 5: MODO CONATIVO (Kathy Kolbe)

- [ ] Kolbe A Index estimado nos 4 modos de acao (Fact Finder / Follow Thru / Quick Start / Implementor)
- [ ] Modo dominante identificado e descrito
- [ ] Estilo de resolucao de problemas natural caracterizado
- [ ] Tensao conativa mapeada (onde o fundador e forcado a agir contra natureza)
- [ ] Recomendacao de ambiente de trabalho ideal baseada no Kolbe

**Se incompleto:** Retornar ao agente kathy-kolbe para aprofundamento.

---

## DIMENSAO 6: ARQUETIPO DE FASCINACAO (Sally Hogshead)

- [ ] Vantagem primaria identificada (Inovacao / Paixao / Poder / Prestigio / Confianca / Misticismo / Alerta)
- [ ] Vantagem secundaria identificada
- [ ] Arquetipo de fascinacao nomeado (entre os 49 arquetipos)
- [ ] Anthem (declaracao de valor percebido) formulado
- [ ] Dormant Advantage identificada (menor vantagem, ponto cego)

**Se incompleto:** Retornar ao agente sally-hogshead para complementar.

---

## CRUZAMENTOS ENTRE FRAMEWORKS

- [ ] Zona de Genialidade (Hendricks) alinhada com Unique Ability (Sullivan)
- [ ] Talentos Clifton coerentes com modo conativo Kolbe
- [ ] Perfil Wealth Dynamics alinhado com estilo de fascinacao Hogshead
- [ ] Oportunidade Hormozi conectada a zona de genialidade real (nao zona de excelencia)
- [ ] Contradicoes entre frameworks documentadas e explicadas
- [ ] Narrativa integradora conecta todas as 6 dimensoes em historia coerente

**Risco critico:** Perfil sem cruzamentos e apenas 6 relatorios separados, nao um mapa integrado.

---

## CLAREZA E ACIONABILIDADE

- [ ] Fundador consegue ler o perfil e se reconhecer imediatamente
- [ ] Linguagem e acessivel (sem jargao tecnico nao explicado)
- [ ] Pelo menos 3 insights "eu nunca tinha visto por esse angulo" presentes
- [ ] Proximos passos concretos estao claros apos leitura
- [ ] Perfil serve como base solida para o Blueprint estrategico
