# Document Quality Checklist

> **Squad:** squad-publisher
> **Purpose:** Validar qualidade de qualquer documento antes de publicacao

---

## 1. Arquitetura de Informacao (Tufte)

- [ ] Hierarquia de informacao definida (primario/secundario/terciario)
- [ ] Informacao agrupada por adjacencia espacial
- [ ] Chartjunk eliminado (sem elementos decorativos sem funcao)
- [ ] Data-ink ratio > 80%
- [ ] Fluxo de leitura logico e natural
- [ ] Small multiples aplicados onde ha series de itens

## 2. Design System (Vignelli)

- [ ] Maximo 2 familias tipograficas
- [ ] Paleta de cores com funcao definida (max 4 cores)
- [ ] Sistema de espacamento baseado em unidade (4pt ou 8pt)
- [ ] Grid definido antes do layout
- [ ] Todos os componentes documentados
- [ ] Consistencia semantica (forma reflete conteudo)

## 3. Grid & Layout (Muller-Brockmann)

- [ ] Grid modular aplicado
- [ ] Margens proporcionais ao formato
- [ ] Gutter consistente em todas as paginas
- [ ] Elementos alinhados ao grid (nenhum fora)
- [ ] Ritmo visual entre paginas consistente
- [ ] Espaco em branco intencional (nao residual)

## 4. Tipografia (Lupton)

- [ ] 3+ niveis de hierarquia visivelmente distintos
- [ ] Contraste adequado entre niveis (minimo 2x diferenca)
- [ ] Measure: 45-75 caracteres por linha
- [ ] Leading: minimo 120% do font size
- [ ] Corpo alinhado a esquerda (nao justificado sem hifenizacao)
- [ ] Contraste de cor: minimo 4.5:1 (AA) para corpo

## 5. Refinamento (Bringhurst)

- [ ] Ritmo vertical: todos os textos alinham na baseline
- [ ] Escala tipografica harmonica (tamanhos nao arbitrarios)
- [ ] Sem orfas (ultima linha isolada no topo)
- [ ] Sem viuvas (primeira linha isolada no fundo)
- [ ] Aspas tipograficas (curvas, nao retas)
- [ ] Travessoes corretos (— nao -)
- [ ] Espacamento simples apos pontos

## 6. Geral

- [ ] Documento funciona em visao macro (folhear rapido) E micro (leitura detalhada)
- [ ] Todas as paginas tem header/footer consistentes
- [ ] Numeracao de pagina na mesma posicao em todas as paginas
- [ ] Dados atualizados e corretos
- [ ] Revisao ortografica/gramatical completa
